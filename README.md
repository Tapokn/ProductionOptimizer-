# ProductionOptimizer-
ProductionOptimizer — C++ программа для оптимизации производственного плана. Определяет оптимальные комбинации выпуска товаров, максимизируя выручку при ограниченных ресурсах сырья и рабочего времени. Выводит все варианты с максимальной прибылью, степень использования ресурсов и обеспечивает отказоустойчивый ввод данных.
